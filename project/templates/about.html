About UnrealiGuana

Introduction

I am an 18-year-old first-year computer science student, who goes by the username UnrealiGuana. I have been coding on an off since the seventh grade. My first experience with coding was some basic Java lessons in school, which I have completely forgotten, because let's face it: Java sucks.

I'm good in Python programming and know some basic C/C++ too. My experience in C++ comes mostly from my pre-university college, which provided C++ education as part of the syllabus, but they taught us in some ancient 16-bit version, which ran on DosBox.

My personal desktop rig is nothing special: It has a singular GTX 1650, an i3-9100f and a 256-GB m.2 SSD. I couldn't get better soecs, partly as I had built it during the chip shortages. My laptop, on the other hand, has a Ryzen 7 5800H and a mobile RTX 3050. Turns out, my laptop is more powerful than my desktop.

I have some experience in Linux as I run Arch on my desktop rig and use a remote Ubuntu server for personal projects. I currently run Windows on my laptop, primarily for compatibility with college software, and partly because I was unable to get the hotkeys to work in anything but Windows.
